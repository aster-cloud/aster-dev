# 5-Minute Quick Start

Aster Lang lets you write **policies, workflows, and decisions** in plain
English (中文, Deutsch, हिन्दी), then run them on a deterministic, auditable engine.

## Your first rule

```aster
Module Greeting.

Rule greet given name:
  return "Hello, " followed by name.
```

Run it in the [Playground](/playground) — every decision is replayable.

## Why Aster

- **Your language** — write rules in the natural language your team speaks.
- **Deterministic** — two runtimes, one IR, identical results.
- **Auditable** — replay any decision; nothing hidden.
