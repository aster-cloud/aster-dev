# 5-Minuten-Schnellstart

Mit Aster Lang schreibst du **Richtlinien, Workflows und Entscheidungen** in
natürlicher Sprache (English, 中文, Deutsch, हिन्दी) und führst sie auf einer
deterministischen, prüfbaren Engine aus.

## Deine erste Regel

```aster
Module Greeting.

Rule greet given name:
  return "Hello, " followed by name.
```

Führe sie im [Playground](/playground) aus — jede Entscheidung ist wiederholbar.

## Warum Aster

- **Deine Sprache** — schreibe Regeln in der natürlichen Sprache deines Teams.
- **Deterministisch** — zwei Laufzeiten, eine IR, identische Ergebnisse.
- **Prüfbar** — jede Entscheidung wiederholbar; nichts versteckt.
